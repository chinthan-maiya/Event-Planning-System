USER MANUAL


  • New user has to register by entering valid credentials
  • Registered user has to login by entering the user name and password to proceed further
  • This is the home page.User details are displayed along with a logout button.
  • 4 options are provided.
    • view previous orders
      • it displays all the events planned and ordered by you
    • plan a new event
      • A page with list of all events is displaayed- Choose one.
      • Now you will be directed to Ordering Page
      • Choose location,date and time. Enter the venue and number of attendees.
      • Choose the requirements from the checkbox list
      • You can check the estimated cost by clicking on get estimate button
      • Confirm your order by clicking on submit button
    • edit previous order
      • you are directed to a page which dislays your previously ordered events
      • choose an event to be edited
      • you are now directed to the event planning & ordering page
      • place the order as earlier
    • feedback-fill the form, write suggestions and comments, submit
  • Every page has a home button and you may logout at any stage
